Select your room reservation CSV files to load the calendar.

Tip: The filename format "XXXX_reservations.csv" will automatically associate the data with the room number XXXX.

Expected CSV Format

Your CSV files should have these column headers:

Display Color,I Want To...,Property,Confirmation Number,External Reference,Name,Alternate Name,Guest Service Status,Membership Type,Membership Level,Membership Number,Block Code,Company,Travel Agent,Source,Group,Reservation Type,Pre Registered,Arrival,Departure,Nights,ETA,ETD,Adults,Children,Rate Code,Rate,Rooms,Room Type,Room Type to Charge,Room,Last Room,Balance,VIP Code,Market Code,Source Code,Linked Name

Important columns used by the calendar:

  • Room - The room number
  • Room Type - Type of room (e.g., STU-BALC)
  • Name - Guest name
  • Arrival - Check-in date in DD.MM.YYYY format
  • Departure - Check-out date in DD.MM.YYYY format
  • Nights - Number of nights
  • Reservation Type - Status of reservation
